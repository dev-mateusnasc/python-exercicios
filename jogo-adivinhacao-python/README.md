# 🎮 Jogo de Adivinhação em Python

Projeto simples desenvolvido durante meus estudos de Python.

O objetivo do jogo é adivinhar um número aleatório entre 1 e 10.

## Funcionalidades

- Geração de número aleatório
- Sistema de tentativas
- Dicas se o número é maior ou menor
- Contador de tentativas

## Tecnologias utilizadas

- Python
- Biblioteca random

## Como executar

1. Baixe o projeto
2. Execute:

python jogo_adivinhacao.py
